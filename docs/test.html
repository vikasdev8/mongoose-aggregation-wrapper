🚀 Mongoose Aggregation Wrapper

✅ GitHub Pages is working! If you can see this page, your deployment is successful.

📦 Installation

npm install mongoose-aggregation-wrapper

🎯 Quick Example

import Wrapper from 'mongoose-aggregation-wrapper'; const results = await Wrapper(UserModel, [ { $match: { active: true } }, { $sort: { createdAt: -1 } }, { $limit: 10 } ]);
